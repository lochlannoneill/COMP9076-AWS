### Running the project

1. Install **'boto3'**
   ```bash
   pip install boto3

2. **(Optional)** Include pre-supplied user information in **'src/config/passwords.txt'** or create new users during runtime.
   ```bash
   David   12345pass   AKIKAOD798  PPB7952+AmaYUd+824nmW
   John    aaaaaaaaa   AKOP67NKAF  CpA9752SDF+709+fa09faAfG
   Joan    bcbcbcddddddd   AIL67NK8NM  KMALF75+95mml+7+89052

3. Execute the application
   ```bash
   python -m src.app
