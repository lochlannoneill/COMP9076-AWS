### Running the project

1. Install **'boto3'**
   ```bash
   pip install boto3

2. Install **'tabulate'** for enhanced terminal output.
   ```bash
   pip install tabulate

3. (Optional) Include user information in **'src/config/passwords.txt'**, see example template below.
